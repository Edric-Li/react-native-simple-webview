# react-native-simple-webview
An auto height webview for React Native


##  Usage is consistent with the official

```javascript
import SimpleWebview from 'react-native-simple-webview';

<SimpleWebview
    scrollEnabled={false}
    originWhitelist={['*']}
    source={{ html, baseUrl: '' }}
    scalesPageToFit={Platform.OS !== 'ios'}
/>
```
